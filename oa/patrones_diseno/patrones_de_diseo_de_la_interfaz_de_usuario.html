Patrones de Diseño de la Interfaz de Usuario

Para el diseño de interfaces de usuario existen ciertos patrones que son los más conocidos.

  • NavegaciónDeAltoNivel .- Se usa cuando un sitio o aplicación implementa cierto número de funciones importantes. Da un menú de alto nivel, acoplado con frecuencia con un logotipo o gráfico identificador que permite la navegación directa hacia cualquiera de las funciones principales del sistema.

 

  • ApilarTarjetas .- Se utiliza cuando deben seleccionarse aleatoriamente cierto número de subfunciones o categorías de contenido específicas relacionadas con una característica o función.

 

  • Llenar los espacios .- Permite introducir datos alfanuméricos en un “cuadro de texto”.
  • OrdenarTabla .- Despliega una lista larga de registros que pueden ordenarse por medio de un mecanismo de cambio para cualquier etiqueta de columna.
  • MigajasdePan .- Brinda una ruta completa de navegación cuando el usuario trabaja con una jerarquía compleja de páginas o pantallas.
  • Editar .- Brinda capacidades de edición de texto sencillo para ciertos tipos de contenido en la ubicación que se muestra en la pantalla.
  • BúsquedaSimple .- Da la capacidad de buscar un sitio web o fuente persistente de datos para un concepto simple de datos descritos por una cadena alfanumérica.
  • Mago .- Lleva al usuario paso a paso a través de una tarea compleja y lo guía para que la termine por medio de ventanas sencillas en la pantalla.
  • CarritodeCompras .- Da una lista de artículos seleccionados para compra
  • IndicadordeAvance .- Proporciona una indicación del avance cuando una operación dura más de n segundos.
