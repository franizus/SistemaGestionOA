Diseño arquitectónico del software

Pregunta

¿Qué es el diseño arquitectónico del software?

Elija una o mas opciones correctas.

Respuestas

El diseño arquitectónico representa la estructura de los datos y de los
componentes del programa que se requieren para construir un
sistema basado en computadora

Es la visión general del sistema, el “plano” de diseño que será la guía
para la construcción del software.

a , b y c

Determina el estilo de arquitectura que adoptará el sistema, la
estructura y las propiedades de los componentes y las interrelaciones
entre ellos.

Pregunta

Los pasos para el diseño arquitectónico son:

1.Comienza con el diseño de los datos

2.Obtención de una o mas representaciones de la estructura arquitectónica del sistema.

3.Se analizan las alternativas de estilos o patrones arquitectónicos para llegar a la estructura mas adecuada para los requerimientos del
usuario y para los atributos de calidad

4.se selecciona una alternativa

5.Se elabora la arquitectura con el empleo de un método de diseño
